# Build it as a static SvelteKit site

A profile page is mostly words and a little style — no login, no database. So we
build it with SvelteKit and export it as static files. Fast to load, cheap to
host anywhere, nothing to keep running.

Decided and settled — this is the ground the plan stands on.
